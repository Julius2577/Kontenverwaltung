# Kontenverwaltung (Java)

Dieses Projekt ist ein einfaches Java-Beispiel zur Verwaltung von Bankkonten.
Es dient als reines Lern- und Übungsprojekt.

## Funktionen
- Anlegen von Konten
- Einzahlen und Abheben
- Überweisungen zwischen Konten
- Ausgabe von Kontoständen

## Technologien
- Java
- Maven
- JUnit (Unit Tests)
- SonarQube for IDE (Codeanalyse)
- Git & GitHub

## Tests
Für zentrale Logik wurden Unit-Tests mit JUnit geschrieben, um die
Funktionalität und Korrektheit sicherzustellen.
